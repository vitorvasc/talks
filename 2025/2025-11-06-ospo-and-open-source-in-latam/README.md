## Abstract

Open source adoption and contribution are growing worldwide, but participation
in [CNCF](https://www.cncf.io/) projects from Latin America (LATAM) remains
disproportionately low in 2025: just 4%. This panel explores how Open Source
Program Offices (OSPOs) can foster sustainable contributor pipelines in LATAM
organizations, and how communities like OpenTelemetry create pathways for new
contributors.
